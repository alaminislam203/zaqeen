# Zaqeen - E-commerce Platform Blueprint

## Overview

Zaqeen is a modern e-commerce platform designed for a seamless shopping experience. It features a client-facing storefront and a comprehensive admin panel for managing products, orders, and site settings.

## Implemented Features

### Storefront

- **Homepage:** Displays a dynamic promotional banner, new arrivals, and featured content.
- **Shop Page:** Lists all available products with filtering and search capabilities.
- **Product Details Page:** Shows detailed information for a single product, including images, description, price, and available sizes.
- **Shopping Cart:** Allows users to add, remove, and update product quantities.
- **Checkout:** A streamlined process for customers to place orders.
- **Contact Page:** A form for customers to send inquiries to the site administrators.

### Admin Panel (`/admin`)

- **Dashboard:** An overview of key metrics (to be implemented).
- **Orders:** Manage and track all customer orders.
- **Products:** A full CRUD (Create, Read, Update, Delete) interface for products, including:
    - Title, description, price, and category.
    - Multi-image uploads.
    - Stock management by size (e.g., S, M, L, XL).
- **Categories:** Manage product categories (e.g., Clothing, Accessories).
- **Customers:** View and manage customer information.
- **Coupons:** Create and manage discount coupons.
- **Messages:** View and manage messages submitted through the contact form.
- **Settings:** A central place to manage global site configurations:
    - **Maintenance Mode:** Toggle the site on/off.
    - **Shipping Fees:** Set delivery charges for inside and outside Dhaka.
    - **Contact Information:** Update the public email and address.
    - **Promotion Banner:** Update the homepage hero banner headline, sub-heading, and background image.

## Current Plan

All initial feature requests have been implemented. The next steps will involve refining existing features, improving the UI/UX, and adding new functionalities as requested.
